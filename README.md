# SYNOPSIS

Sparrowdo module to install/update Sparrow toolchain on target host.

# Install

    $ panda install Sparrowdo::Sparrow::Update


# Usage

    $ cat sparrowfile

    use v6;

    use Sparrowdo;

    module_run 'Sparrow::Update';
